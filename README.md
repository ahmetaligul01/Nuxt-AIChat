### Setup

## Running Locally

### 1. Clone Repo
```bash
git clone https://github.com/ahmetaligul01/Nuxt-AIChat
```

### 2. Install Dependencies
```bash
# npm
npm install
```

### 3. Run Ollama server
[desktop client](https://ollama.com/download)

### 4. Run App (Development Server)

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
